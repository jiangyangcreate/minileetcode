## 简介

这个项目包含了一系列常见的 LeetCode 算法问题的解答，每个问题都有对应的 Python 解决方案。通过这个项目，你可以学习到不同问题的解题思路和算法实现。

## 如何使用

1. 克隆或下载项目到本地：

   ```bash
   git clone https://github.com/your-username/mini-leetcode.git
   ```

2. 进入项目目录：

   ```bash
   cd mini-leetcode
   ```

3. 使用你喜欢的编辑器打开代码并开始学习和练习。

## 项目结构

```plaintext
mini-leetcode/
|-- problems/
|   |-- problem1.py
|   |-- problem2.py
|   |-- ...
|-- tests/
|   |-- test_problem1.py
|   |-- test_problem2.py
|   |-- ...
|-- README.md
|-- .gitignore
|-- LICENSE
|-- ...
```

- `problems/`: 包含每个算法问题的 Python 解决方案文件。
- `tests/`: 包含每个算法问题的测试文件，用于验证解决方案的正确性。
- `.gitignore`: Git 忽略文件，用于指定不需要纳入版本控制的文件。
- `LICENSE`: 项目的许可证文件。
